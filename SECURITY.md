# Security Policy

## Supported releases

Hatui is currently pre-1.0. Security fixes are applied to the latest released version only.

## Reporting a vulnerability

Please do not open a public issue for suspected vulnerabilities.

Instead, report the issue privately to the project maintainers through the security reporting channel configured for the repository. Include:

- affected version
- impact summary
- reproduction details
- any suggested mitigation

You should receive an acknowledgment within a reasonable time after the report is received.
